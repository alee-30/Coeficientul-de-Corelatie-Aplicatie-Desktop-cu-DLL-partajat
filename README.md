Coeficientul de Corelație - Aplicație Desktop cu DLL Partajat

Acest proiect reprezintă o aplicație desktop (C++) dezvoltată pentru disciplina Sisteme de Operare de la facultate. 
Obiectivul principal este calculul coeficientului de corelație R între două seturi de date, utilizând o bibliotecă de link-editare dinamică (DLL) pentru procesare și partajarea datelor.

Funcționalități
- Calcul Matematic: Implementarea formulei de corelație Pearson pentru variabile discrete
- Arhitectură Modulară: Logica de calcul este izolată într-un fișier .dll
- Partajarea Datelor: Permite partajarea valorilor numerice între mai multe instanțe ale aplicației prin intermediul memoriei partajate în DLL

Tehnologii Utilizate
- Limbaj: C++ / C# 
- IDE: Visual Studio 2022
- Concepte SO: Dynamic Link Library (DLL), Shared Data Segments, Gestiunea excepțiilor

In fisierul Proiect Individual-Nr.20 se afla niste capturi de ecran cu rezultatele obtinute in urma rularii.
